Getting Started: Chromium Container\n1. Add & Connect your container\n* Add a **Chromium Container**\n* Connect: `Chromium eth0 → Switch1`\n2. Configure networking\n* Right‑click the container → **Configure**\n* Under **Network configuration**, click **Edit**\n* Enable DHCP for `eth0` (un‑comment DHCP lines and set hostname if desired)\n* Enable Static IP for eth0 (un‑comment static IP lines and set hostname if desired)\n3. **Save & Start**\n* Click **Save** → **OK**\n* Right‑click → **Start**\n4. **Launch Browser**\n* Right‑click → **Console** or double‑click the container\n
